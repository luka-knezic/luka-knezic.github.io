Little bit about me

I first started programming as a hobby in winter of 2014 when I made a simple console application that reads computer specifications.

Since then, my curiosity and passion for more got the better of me, and I always find myself looking for something to tinker with.

My profile

Throughout my education as well as my personal hobbies, I've come to learn a range of programming (and scripting) languages. My most used is C#, as well as PHP (and MySQL) for the backend side of things. I'm also no stranger to the likes of JavaScript and C++.

Along with my programming arsenal, I've used a wide range of accompanying software. My love for gaming paved my way as a hobby game dev with Unity, and the fact it uses C# also prompted me to start using Xamarin for developing Android applications. This in turn pushed me into backdev and its partner-in-crime database management. I'm also familiar with source control like Git, or more specifically GitHub (as evident by this page itself).
As far as code editors go, VS Code is among my favourite ;)

I'm a native Croat, but I am also very fluent in English. I know a few basics in German too.
